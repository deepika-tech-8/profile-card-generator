# User Profile Card Generator

A simple and interactive **User Profile Card Generator** built using HTML, CSS, and JavaScript.  
This project allows users to create a dynamic profile card with a live preview and download it as an image.

## 🚀 Features

-   Create a profile card using user input
-   Live preview while typing
-   Upload a profile image from your computer
-   Add social media links (LinkedIn, GitHub, Instagram)
-   Download the profile card as an image
-   Dark / Light mode toggle
-   Clean and responsive UI

## 🛠️ Technologies Used

-   HTML
-   CSS
-   JavaScript
-   html2canvas (for downloading the card as an image)
